# Automatas celulares de trafico mixto

Notebook con la implementacion del modelo de automata celular para trafico mixto (autos y motos) con duracion de cambio de carril. Incluye:

- Marco teorico y parametros del modelo.
- Simulador en C++ incrustado en el notebook (pensado para Colab).
- Graficas de flujo y velocidad vs densidad.
- Comparacion cualitativa y cuantitativa con el articulo.

## Uso rapido (Colab)

1. Abre Ag.ipynb en Google Colab.
2. Ejecuta la celda de C++ para generar results.csv.
3. Ejecuta la celda de graficas y la de comparacion cuantitativa.
